# Quick Reference Checklists for Powered Parachute Operations

---

## Pre-Flight Inspection Checklist

### External Inspection (Clockwise from Nose Wheel)

**Nose Wheel**
- [ ] Tire pressure 8-12 psi
- [ ] No damage to tire
- [ ] Wheel secure
- [ ] Brake functional
- [ ] Steering operates smoothly

**Front Left**
- [ ] Steering bar secure
- [ ] Steering lines good condition
- [ ] Trim system set correctly
- [ ] No line tangles

**Instrument Panel**
- [ ] All instruments functional
- [ ] Switches OFF
- [ ] Key removed or secured

**Left Seat**
- [ ] Seat belt intact
- [ ] Buckle operates
- [ ] Seat secure to frame

**Left Riser**
- [ ] Carabiner locked
- [ ] Quick link tight (finger tight + 1/4 turn)
- [ ] Riser in good condition
- [ ] No twists or tangles

**Left Wheel**
- [ ] Tire pressure correct
- [ ] Tire condition good
- [ ] Axle nut secure with cotter pin

**Left Engine Area**
- [ ] Battery secure, no leaks
- [ ] Coolant level OK
- [ ] No leaks
- [ ] Radiator secure

**Left Side Engine**
- [ ] Exhaust secure, no cracks
- [ ] Springs intact
- [ ] Spark plugs tight
- [ ] No oil leaks

**Propeller**
- [ ] Clean (no bugs/debris)
- [ ] No damage
- [ ] All bolts tight
- [ ] No wobble

**Prop Cage**
- [ ] All tubes secure
- [ ] No cracks
- [ ] Proper clearances

**Gearbox**
- [ ] No leaks
- [ ] Oil level correct

**Right Engine**
- [ ] Carburetors secure
- [ ] No cracks in boots
- [ ] Fuel lines secure
- [ ] Throttle linkage smooth

**Fuel System**
- [ ] Fuel cap tight
- [ ] Fuel level adequate
- [ ] Gascolator drained (no water)
- [ ] No leaks

**Right Rear**
- [ ] Oil tank full
- [ ] Rotary valve oil full
- [ ] Coolant overflow OK
- [ ] No leaks

**Right Wheel**
- [ ] Tire pressure correct
- [ ] Condition good
- [ ] Axle secure

**Right Riser**
- [ ] Carabiner locked
- [ ] Quick link tight
- [ ] Riser good condition

**Right Front**
- [ ] Throttle operates smoothly
- [ ] Friction adjusted
- [ ] Primer functional

**Final**
- [ ] Area clear
- [ ] Covers removed
- [ ] Nothing left on ground

## Engine Start Checklist

### Before Start
- [ ] Aircraft on level ground
- [ ] Wing secured or helpers ready
- [ ] Prop area clear of people
- [ ] Covers removed
- [ ] Oil levels checked (both)
- [ ] Coolant level checked
- [ ] Fuel sufficient
- [ ] Gascolator drained

### Starting (Warm Engine)
- [ ] Mags OFF
- [ ] Master ON
- [ ] Prime 2-3 shots
- [ ] Throttle cracked 1/4"
- [ ] Yell "CLEAR PROP!"
- [ ] Look around
- [ ] Turn key to START
- [ ] Release key when starts
- [ ] Idle at 2,000 RPM

### Starting (Cold Engine)
- [ ] Same as warm start
- [ ] Prime 4-6 shots
- [ ] Apply choke if equipped
- [ ] May need longer cranking
- [ ] Warm at 2,500 RPM

## Engine Warm-Up and Run-Up Checklist

### Warm-Up (5+ Minutes)
- [ ] Start at low idle
- [ ] Gradually increase to 3,000-3,500 RPM
- [ ] Vary RPM (don't hold constant)
- [ ] Monitor all temperatures
- [ ] Coolant temp 140°F minimum
- [ ] Voltage 13.5-14.5V

### Mag Check
- [ ] Secure aircraft
- [ ] 3,500 RPM
- [ ] Left mag OFF
- [ ] Note drop (100-300 RPM acceptable)
- [ ] Left mag ON
- [ ] Right mag OFF
- [ ] Note drop (100-300 RPM acceptable)
- [ ] Right mag ON
- [ ] Return to idle

### Full Power Check
- [ ] Secure aircraft (tie down or strong helper)
- [ ] Sit in seat, hands on mags
- [ ] Engine at operating temp
- [ ] Full throttle
- [ ] Check RPM (6,350-6,450 typical)
- [ ] All temps normal
- [ ] Duration 5-10 seconds only
- [ ] Return to idle

## Wing Layout Checklist

### Site Selection
- [ ] Flat, clear area
- [ ] Into the wind
- [ ] Free of obstacles
- [ ] Adequate takeoff distance

### Layout
- [ ] Bag behind aircraft, label away
- [ ] Remove wing from bag
- [ ] Lay out (stacked or spread method)
- [ ] Remove line socks
- [ ] Separate cells if needed

### Line Check (CRITICAL)
- [ ] Start at risers
- [ ] Follow EVERY line to wing
- [ ] Check for tangles
- [ ] Check for line-overs
- [ ] Check for twists
- [ ] Check steering lines through pulleys
- [ ] Check both sides completely
- [ ] Take up riser/steering slack
- [ ] Final check - walk through again

### Wing Inspection
- [ ] Walk length of wing
- [ ] Check fabric condition
- [ ] Check for tears
- [ ] Check leading edge openings
- [ ] Check trailing edge
- [ ] Verify cells will inflate

## Takeoff Checklist

### Lineup
- [ ] Directly into wind
- [ ] Runway/area clear
- [ ] Wing properly positioned

### Takeoff Sequence
1. [ ] Feet slightly forward on steering bars
2. [ ] Smooth full throttle
3. [ ] Wing begins to rise
4. [ ] Minor corrections only
5. [ ] Glance up - wing overhead?
6. [ ] Wing looks good?
7. [ ] Continue accelerating
8. [ ] Aircraft feels light
9. [ ] Gentle liftoff
10. [ ] Establish positive climb

### Reject Takeoff If:
- [ ] Wing doesn't inflate properly
- [ ] Lines visible
- [ ] Wing unstable
- [ ] Obstacle appears
- [ ] Wind shift
- [ ] Engine problem
- [ ] "Bad feeling"

---

## Landing Checklist

### Landing Sequence
1. [ ] Short final - maintain approach speed
2. [ ] 15-20 feet - begin flare
3. [ ] Reduce power gradually
4. [ ] Continue flare
5. [ ] 5-10 feet - full flare
6. [ ] Power idle before touchdown
7. [ ] Main wheels touch
8. [ ] Nose wheel touches gently

### Immediately After Touchdown
1. [ ] Full forward on steering bars
2. [ ] Pull steering lines in by hand
3. [ ] Wing collapses behind aircraft
4. [ ] Apply brake if needed
5. [ ] MAG SWITCHES OFF

## Post-Flight Checklist

### Shutdown
- [ ] Mags OFF (engine stops)
- [ ] Master OFF
- [ ] Key removed
- [ ] Wait for prop to stop
- [ ] Remain seated until prop stopped

### Post-Flight Inspection
- [ ] Walk around for damage
- [ ] Check for leaks
- [ ] Check tires
- [ ] Brief wing inspection

### Securing Aircraft
- [ ] Pack wing in bag
- [ ] Secure bag to aircraft
- [ ] Install covers (prop, exhaust, carbs)
- [ ] Fuel cap verified tight
- [ ] Tie down if leaving outside

### Servicing
- [ ] Refuel (if needed)
- [ ] Fill oil tank
- [ ] Check coolant level
- [ ] Log flight time
- [ ] Note any issues
- [ ] Clean up area

## Emergency Checklists

### Engine Failure In Flight
1. [ ] Fly the aircraft - maintain control
2. [ ] Establish best glide speed (~25 mph)
3. [ ] Select landing area
4. [ ] Attempt restart (if altitude permits)
5. [ ] Commit to landing
6. [ ] Plan approach
7. [ ] Execute approach
8. [ ] Full flare before touchdown
9. [ ] Collapse wing after landing

### Engine Failure on Takeoff

**Before Liftoff:**
1. [ ] Close throttle immediately
2. [ ] Mags OFF
3. [ ] Full forward steering (collapse wing)
4. [ ] Brake
5. [ ] Assess problem

**Just After Liftoff (0-50 feet):**
1. [ ] Land straight ahead - DON'T TURN
2. [ ] Maintain control
3. [ ] Flare normally
4. [ ] Collapse wing after touchdown

**Higher Altitude (50+ feet):**
1. [ ] Fly aircraft - maintain control
2. [ ] Land ahead (slight turns OK)
3. [ ] Best available field
4. [ ] Standard approach if possible
5. [ ] Commit to landing

### Fire In Flight
1. [ ] Mags OFF (shut down engine)
2. [ ] Dive (increase airspeed - may extinguish)
3. [ ] Land IMMEDIATELY
4. [ ] Evacuate as soon as stopped
5. [ ] Get away from aircraft

### Wing Collapse
1. [ ] Release brakes/steering - LET GO!
2. [ ] Allow wing to recover
3. [ ] Don't fight it
4. [ ] Maintain airspeed
5. [ ] Land as soon as practical

## Quick Emergency Reference

### Emergency Priority
1. **FLY THE AIRCRAFT** (always first)
2. Analyze the situation
3. Take appropriate action
4. Land as soon as practical/possible

### Emergency Landing Site Selection
**Good:** Large open fields, into wind, level, clear approach
**Avoid:** Trees, power lines, buildings, water, rough terrain

### After Emergency Landing
1. Secure aircraft (mags off, master off)
2. Check for injuries
3. Evacuate if fire risk
4. Call for help (911)
5. Stay with aircraft if safe

### Emergency Contacts
- **911**: Emergency services
- **1-800-WX-BRIEF**: Weather
- **NTSB**: 800-877-6799 (accident reporting)
